# PUP Hymn 

A single-page web project featuring a synchronized lyric player for the Polytechnic University of the Philippines (PUP) Hymn.

## Features
- **Synchronized Lyrics**: Lyrics automatically highlight and scroll in sync with the audio.
- **Playback Controls**: Play, pause, and seek through the hymn.
- **Speed Adjustment**: Control the playback speed (0.5x to 3.0x).
- **Volume Control**: Easily adjust the audio volume.

## Tech Stack
- HTML
- CSS
- JavaScript
